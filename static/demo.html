Built for Hermes × NemoClaw × Stripe × Blockrun × NVIDIA NIM

    
Built for the Hermes Accelerated Agent Business Hackathon — this is a live demo. The evaluator field and Stripe dashboard are both verifiable now.

    

      

        
Hermes

        
the agent orchestrator. This demo simulates a Hermes agent selecting tools, triggering payments via Stripe Skills, and hitting Beacon's escrow gate before any money moves.

      

      

        
NemoClaw

        
the orchestration framework the agent runs on. NemoClaw is a hackathon partner and execution runtime — it doesn't appear in the Agent Directory because it isn't a scored, consumable API.

      

      

        
Stripe Skills for Hermes

        
dual-rail payments: Stripe PaymentIntents (high-value escrow) or Blockrun x402 (agent-native micropayments). Real test-mode PaymentIntents with manual capture, or gasless EIP-3009 USDC settlement on Base. The escrow gate blocks bad endpoints entirely, on either rail.

      

      

        
Blockrun

        
settles the x402 leg. Once escrow clears, Blockrun settles USDC on Base for the agent's completed request.

      

      

        
NVIDIA NIM

        
the real scored API in this demo. A live probe against integrate.api.nvidia.com is what the agent actually pays to call.

      

      

        
NVIDIA Nemotron 3 Ultra

        
the scoring brain. Live inference via Nous Research API. Every probe gets a real LLM evaluation, not a heuristic.
